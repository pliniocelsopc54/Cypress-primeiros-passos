# My First Cypress Automation
this project utilizes Cypress to automate the testing of a login feature and a user update feature. Cypress is a Javascript-based end-to-end testing framework that allows developers to write tests and execute them a real browser.

## instalation
```bash
 npm install
 ```
>**NOTE:**
>
> it's necessary to have **node** installed

## Runnig
```bash
# You open the Cypress UI
npx cypress open

# Run the automation by command line
npx cypress run
```
